# Claude Cowork 요약 재요청 프롬프트

기존 엑셀의 summary 컬럼이 '본문 복사'라서, **진짜 요약만** 다시 받기 위한 요청.
저자·소속·본문은 이미 정확하므로 건드리지 않고, 요약만 새로 받아 갈아끼운다.

---

## 붙여넣을 프롬프트

```
C:\VS_Test\tech_insight\data_source\정보과학회지 폴더의 각 논문 PDF를 읽고,
논문별 '진짜 요약'을 만들어 엑셀(.xlsx) 하나로 만들어줘.

[중요 — 이전 요약의 문제점]
이전에 만든 요약은 본문 첫 문단을 그대로 복사한 것이었어. 이번엔 그러지 말 것.
본문을 읽고 내용을 이해해서 새로 써야 해.

[요약 작성 기준]
- 한국어로 3~5문장.
- 다음을 담을 것: ① 이 논문이 다루는 주제/문제, ② 핵심 방법이나 접근,
  ③ 주요 결론·기여·전망.
- 본문 문장을 그대로 옮기지 말고, 내용을 소화해서 새 문장으로 작성.
- 도입부 인사말·배경 나열이 아니라 "이 논문의 핵심"을 요약.

[엑셀 컬럼 — 정확히 이 2개, 영문 헤더]
1. file_path : data_source 기준 상대경로
               (예: 정보과학회지\26_5\산업별 AGI 기술 활용...pdf)
               ※ 이전 엑셀과 동일한 file_path 값을 그대로 써야 매칭됨
2. summary   : 위 기준으로 새로 작성한 요약

[형식]
- 첫 행은 헤더(file_path, summary).
- 한 논문 = 한 행.
- 행정성 문서(목차·학회동정·특집계획 등)는 제외.
- 결과 파일명: tech_insight_summaries.xlsx
```
